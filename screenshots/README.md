# Screenshots

This directory contains screenshots of the Developer Velocity Dashboard.

## How to add screenshots:

1. Take screenshots of:
   - Main dashboard view (logged in state)
   - Login/authentication screen
   - Task management interface
   - Weekly analytics chart

2. Save as:
   - `dashboard.png` - Main dashboard screenshot
   - `login.png` - Login screen screenshot
   - `analytics.png` - Analytics/charts view (optional)

3. Recommended dimensions: 1200x800px or similar 16:10 ratio
